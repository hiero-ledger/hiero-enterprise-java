# Hiero Enterprise Java — Technical Documentation

Welcome to the technical documentation for the **Hiero Enterprise Java**.

Hiero Enterprise Java provides Java modules to interact with a [Hiero network](https://hiero.org) in Spring Boot or Eclipse MicroProfile (e.g. Quarkus) applications.

---

## Documentation Structure

- **[Getting started](getting-started.md)** — How to add and configure Hiero Enterprise in your project.
- **[Architecture](architecture.md)** — High-level structure and components.
- **[Spring Boot](spring-boot.md)** — Spring Boot integration and usage.
- **[MicroProfile](microprofile.md)** — MicroProfile / Quarkus integration and usage.
- **[Clients & Repositories](client-and-repository.md)** — Core clients and repositories services (AccountClient, FileClient, etc.).
